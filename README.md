# ✨ Kani's Magic Dust (White Day Surprise)

A magical, interactive particle animation built with HTML5 Canvas and Vanilla JavaScript. Designed specifically as a creative and lightweight web experience for a White Day surprise.

## 🌟 Features
* **Interactive Canvas:** Particles react and transition based on user clicks or screen taps.
* **Smooth Transitions:** Custom physics (friction, velocity, and color lerping) for seamless particle movements.
* **Text-to-Particle Rendering:** Converts text and emojis (🦀) into dynamic particle formations.
* **Responsive Design:** Automatically adjusts particle density and font sizes for both desktop and mobile screens.
* **Immersive Audio:** Includes background music that triggers upon the first user interaction.
